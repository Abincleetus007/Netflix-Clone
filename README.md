# Netflix Clone 

This is a Netflix clone web application built with React, CSS, and integrated with The Movie DB (TMDb) API. It provides a user-friendly interface where users can browse, search, and explore movies and TV shows.

## Features
- Display trending movies and TV shows
- Search functionality for movies and TV shows
- View movie details including title, overview, and release date
- Responsive design
- Based on The Movie DB API for fetching movie data

## Tech Stack
- **Frontend:** ReactJS, CSS (Flexbox/Grid)
- **API:** The Movie DB API
- **State Management:** React Hooks (useState, useEffect)
